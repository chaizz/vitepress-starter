# Obsdian插件

直接在obsdian的插件面板直接搜索想要的插件相关关键字。

>s在插件面板中搜索 markdown转化为world文件类型。



插件推荐：

1. 统计笔记的数量：File Explorer Note Count

2.  最近访问的笔记：Recent File

3. Markdown 转化为 World 文件类型：Pandoc

4. 将大纲内容转化为思维导图形式： Mindmap

5. 绘制手绘风格流程图：Excaldraw
